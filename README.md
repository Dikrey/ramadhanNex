## ğŸ“– Tentang Ramadan Elite

**Ramadan Elite** adalah platform dashboard ibadah digital berbasis web yang dirancang untuk mengoptimalkan pengalaman spiritual selama bulan suci Ramadan. Menggabungkan estetika _Glassmorphism_ modern dengan fungsionalitas tingkat tinggi, aplikasi ini membantu pengguna melacak progres ibadah, jadwal sholat, hingga manajemen kesehatan (water intake & sleep tracker) secara real-time.

Dibangun dengan teknologi **Vanilla JavaScript** untuk performa maksimal dan **Tailwind CSS** untuk antarmuka yang responsif, Ramadan Elite menawarkan fitur unggulan seperti _Audio Adzan Player_ otomatis, _Streak System_ berbasis mood, dan _Notification Center_ terpadu untuk memastikan pengguna tetap istiqomah dalam beribadah.
